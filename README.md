- 📍 **Search by keyword + location** (e.g., "plumbers in Dallas")

- 📁 **CSV export** — ready for your CRM or cold email campaign
- 🔄 **Unlimited results** — no pagination limits, no daily caps
- 🛡️ **No Google API key needed** — works via public Google Maps interface
